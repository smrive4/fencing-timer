# Fencing Timer

A web-based timer application designed specifically for fencing bouts and training sessions.

**Live Demo:** [https://fencing-timer.vercel.app/](https://fencing-timer.vercel.app/)

## Features

- **Precision Timing** - Accurate bout timing for competitive fencing
- **Multiple Formats** - Support for different fencing disciplines (Foil, Épée, Sabre)
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Instant Controls** - Quick start, stop, and reset functionality

## Usage

1. Visit [https://fencing-timer.vercel.app/](https://fencing-timer.vercel.app/)
2. Use the controls to start, pause, and reset the timer

## Technology Stack

- **Frontend Framework:** [React]
- **Deployment:** Vercel

## Fencing Bout Rules

This timer follows official FIE (Fédération Internationale d'Escrime) timing rules:

- **Standard Bout:** 3 minutes of fencing time per period
